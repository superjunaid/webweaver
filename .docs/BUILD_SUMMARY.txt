================================
QUICK START (INSTALLATION)
================================

1. Copy plugin folder to /wp-content/plugins/webweaver/
2. Activate via WP Admin → Plugins
3. Go to WP Admin → WebWeaver → Settings
4. Configure options (draft-only is ON by default)
5. Generate Application Password (Users → Your Profile)
6. Use API with Basic Auth: Authorization: Basic base64(username:password)
7. Check Tools Manifest: GET /wp-json/wp-mcp/v1/tools
